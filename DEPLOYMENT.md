# AWS Community Day - Deployment Guide\n\nA comprehensive guide to deploy the AWS User Route - Madurai Cloud Community Day website.\n\n## üöÄ Quick Deployment\n\n### Prerequisites\n- Node.js 16+ installed\n- npm or yarn package manager\n- Git\n\n### Local Development\n\n1. **Clone the repository**\n   ```bash\n   git clone <repository-url>\n   cd aws-ug-mdu-Community-day\n   ```\n\n2. **Install dependencies**\n   ```bash\n   npm install\n   ```\n\n3. **Start development server**\n   ```bash\n   npm start\n   ```\n\n4. **Open browser**\n   Navigate to [http://localhost:3000](http://localhost:3000)\n\n### Production Build\n\n```bash\nnpm run build\n```\n\nThis creates an optimized production build in the `build/` directory.\n\n## üåê Deployment Options\n\n### 1. Netlify (Recommended)\n\n**Automatic Deployment:**\n1. Connect your GitHub repository to Netlify\n2. Set build command: `npm run build`\n3. Set publish directory: `build`\n4. AWS S3 + CloudFront\n\n1. **Build the project**\n   ```bash\n   npm run build\n   ```\n\n2. **Create S3 bucket**\n   ```bash\n   aws s3 mb s3://aws-community-day-madurai\n   ```\n\n3. **Upload files**\n   ```bash\n   aws s3 sync build/ s3://aws-community-day-madurai --delete\n   ```\n\n4. **Configure bucket for static hosting**\n   ```bash\n   aws s3 website s3://aws-community-day-madurai --index-document index.html --error-document index.html\n   ```\n\n5. **Set up CloudFront distribution** (optional but recommended)\n\n### 4. GitHub Pages\n\n1. **Install gh-pages**\n   ```bash\n   npm install --save-dev gh-pages\n   ```\n\n2. **Add to package.json**\n   ```json\n   {\n     \"homepage\": \"https://yourusername.github.io/aws-ug-mdu-Community-day\",\n     \"scripts\": {\n       \"predeploy\": \"npm run build\",\n       \"deploy\": \"gh-pages -d build\"\n     }\n   }\n   ```\n\n3. **Deploy**\n   ```bash\n   npm run deploy\n   ```\n\n## ‚öôÔ∏è Configuration\n\n### Environment Variables\n\nCreate a `.env` file in the root directory:\n\n```env\n# Optional: Google Maps API key for enhanced location features\nREACT_APP_GOOGLE_MAPS_API_KEY=your_api_key_here\n\n# Optional: Analytics tracking ID\nREACT_APP_GA_TRACKING_ID=your_tracking_id_here\n\n# Optional: Social media API keys for live feed\nREACT_APP_TWITTER_API_KEY=your_twitter_api_key\nREACT_APP_LINKEDIN_API_KEY=your_linkedin_api_key\n```\n\n### Customization\n\n#### Event Data\nEdit `src/App.js` to modify:\n- Event details (name, date, venue)\n- Speaker information\n- Schedule items\n- Sponsor logos\n- FAQ content\n\n#### Styling\nCustomize colors and themes in `src/App.css`:\n```css\n:root {\n  --aws-orange: #FF9900;\n  --aws-blue: #232F3E;\n  /* Add your custom colors */\n}\n```\n\n#### Components\nAll components are modular and can be easily customized:\n- `src/components/Hero.js` -
